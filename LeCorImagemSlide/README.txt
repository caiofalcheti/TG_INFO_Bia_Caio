Este código apresenta a cor que está abaixo do cursor em uma imagem. As imagens podem avançar com a tecla "N" e voltar com a tecla "P".
Este código foi desenvolvido na disciplina "Processamento de Vídeo" com o auxílio do Prof. Celso.
A ideia é incrementá-lo para que, após coletar os parâmetros de cor, com um clique, um som será tocado a partir dos parâmetros.
